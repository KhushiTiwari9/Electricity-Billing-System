# ⚡ Electricity Billing System

A Java-based desktop application designed to simplify electricity bill management for both customers and administrators. This project automates billing operations, customer record management, bill generation, and payment tracking through an interactive GUI.

## 🚀 Features

* 👤 Customer Registration & Management
* 🔐 Secure Login & Signup System
* ⚡ Meter Information Management
* 🧾 Automatic Bill Calculation
* 💳 Bill Payment Module
* 📄 Payment Receipt Generation
* 📊 Deposit & Billing Details Tracking
* 🎨 User-Friendly Java Swing Interface
* 🗄️ MySQL Database Integration

## 🛠️ Tech Stack

* **Language:** Java
* **GUI:** Java Swing
* **Database:** MySQL
* **Build Tool:** Maven
* **IDE:** NetBeans / IntelliJ IDEA

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone <your-repository-link>
```

2. Open the project in your preferred Java IDE.

3. Import the database:

* Open MySQL
* Import the `database.sql` file

4. Configure database credentials inside:

```java
Conn.java
```

5. Run the project.
